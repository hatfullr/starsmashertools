starsmashertools.lib.table

class TEOS(*args, verbose: bool = False, **kwargs)[source]

The tabulated equation of state (TEOS) used in a StarSmasher simulation. Use interpolate() to obtain values from the TEOS. Note that this class is equipped only to read a specific table format. If you are experiencing troubles with your table’s format, you may need to create your own file reading protocol, likely by creating a subclass. Implement your subclass by creating an instance of your subclass and then setting the _teos attribute of your simulation to that instance. The next time you access the teos attribute, it should return your subclass instance.

Parameters:

  • rho (float, list, tuple, numpy.ndarray) – Density values for interpolation, in cgs units.

  • u (float, list, tuple, numpy.ndarray) – Specific internal energy values for interpolation, in cgs units.

  • which (str, int) – The value to obtain by interpolation. If a str is given, it must correspond to one of the table column headers. If a int is given it is interpreted as a column number.
