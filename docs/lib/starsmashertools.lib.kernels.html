class _BaseKernel(compact_support: int | float = 2, tabulated: bool = False, ntab: int = 100, integrated: bool = False, nstp: int = 100, interpolate_table: bool = True)[source]ÔÉÅ

An SPH kernel function in 3D.

Parameters:

  • compact_support (int, float, default = 2) ‚Äì Particle smoothing length \(h\) will be multiplied by this number to implement compact support.

  • tabulated (bool, default = False) ‚Äì If True, then values of this function will be tabulated for quick lookup. The number of tabulated values is ntab.

  • ntab (int, default = 1000) ‚Äì If tabulate is True, this many values will be stored in memory for quick lookup.

  • integrated (bool, default = False) ‚Äì If True, this kernel function will be the 2D integrated kernel function. See set_integrated() for more details. This will automatically make the kernel function be tabulated.

  • nstp (int, default = 100) ‚Äì If integrated is True, then this many integration steps are used.

  • interpolate_table (bool, default = True) ‚Äì If True and if this kernel function is tabulated, calculations will be done by linear interpolating across the table. If False, then calculations will be done by simply finding the closest value, without interpolating.

set_integrated(value: bool, nstp: int = 100)[source]ÔÉÅ

Set this kernel function to be an integrated kernel function if it isn‚Äôt one already, or a non-integrated kernel function if it is. An integrated kernel function has only 2 dimensions instead of 3. The integrated form is as though the 3D form were squished down into a flat disk. The distance \(x\) used in scaled() and __call__() becomes a 2D distance.

Parameters:

  • value (bool)

  • nstp (int, default = 100) ‚Äì The number of integration steps to use.
