starsmashertools.mpl.figure

class FluxFigure(*args, values: list | tuple = ['flux'], **kwargs)[source]

Show any number of FluxResults on a Figure. The figure appears as a collection of images, where each row represents one FluxResult. The number of columns in each row is determined by the keys given in the constructor.

Parameters:

  • *args – Positional arguments are passed directly to __init__().

  • values (list, tuple, default = Pref('values', ['flux'])) – A list of str or functions which determine the quantities to be shown. Each str must correspond with a key in the 'image' dict of the FluxResult. If a function is given, it must accept a FluxResult as input and give a 2D numpy.ndarray and a dict as the output, where the 2D array is the image and the dict specify keyword arguments to matplotlib.axes.Axes.imshow().

  • **kwargs – Other keyword arguments are passed directly to __init__().

add(result: str | FluxResult)[source]

Add a FluxResult to the figure.

Parameters:

  • result (str, FluxResult) – The FluxResult to add to the figure. If a str is given, it must be a path to a valid FluxResult file.

  • values (list, tuple, None, default = None) – The str keys in the ‘image’ dict of the FluxResult which will be plotted.
