get_binding_energy(output: Output | OutputIterator, mass_coordinate: int | float | None = None)[source]

Returns the binding energy of the star’s envelope \(E_\mathrm{SPH}\) at some specified mass coordinate \(m(r)\), as defined in equation (12) of Hatfull et al. (2021).

Parameters:

  • output (Output, OutputIterator) – If a Output is given, the binding energy will be calculated just for that output file and a single value will be returned. If a OutputIterator is given then the binding energy will be calculated for all Output objects in the iterator and a list of values will be returned.

  • mass_coordinate (int, float, None, default = None) – The mass coordinate \(m(r)\) to calculate the binding energy at. Set to None to get the total binding energy of the envelope, skipping the core particle if there is one. That is, \(m(r) = m_{core\ particle}\). Set to 0 to get the binding energy of the entire star, including any core particles (perhaps not valid?). A ValueError is raised if this argument is negative.

Returns:

If output is of type Output then a Unit is returned in cgs units. If output is of type OutputIterator then a list of Unit objects is returned.

Return type:

Unit or list

get_core_particles(), Unit
